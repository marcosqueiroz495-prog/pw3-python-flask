Sistema de registro e avaliação de livros

A Biblioteca Avaliativa é uma aplicação web destinada ao registro e organização de obras literárias. O sistema permite cadastrar livros, atribuir avaliações e visualizar os registros realizados pelos usuários.

Este projeto foi desenvolvido para demonstrar conceitos fundamentais de desenvolvimento web utilizando a arquitetura MVC e a tecnologia Flask.
